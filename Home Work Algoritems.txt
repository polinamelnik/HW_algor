1. Нужно создать переменную "maxNumber", со значением "-Infinity", в которой будет находиться максимальное число.
2. Нужно пройтись по всем элементам массива и сравнить его со значением "maxNumber". Если текущий элемент больше нашей переменной,то значит нужно обновить наше значение на текуший элемент.
3. После того как прошлись по всем элементам массива, нужно вернуть значение "maxNumber" как максимальное значение

Обьяснение:
Оценка временной сложности данного алгоритма составляет O(n), где n - размер входного массива. В худшем случае, алгоритм должен просмотреть все элементы массива один раз, чтобы определить максимальное число. Нет возможности сократить количество операций, поскольку в неотсортированном массиве максимальное число может находиться в любой позиции. Таким образом, алгоритм работает линейно относительно размера массива.
